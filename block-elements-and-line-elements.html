Elementos de bloque y elementos de línea

¿Qué son los elementos de bloque?

En pocas palabras, son aquellos elementos que ocupan todo el espacio que está a su derecha, un poco arriba y un poco abajo, así el contenido de ellos no lo requiera.

Etiquetas que se comportan como elementos de bloque

  • <p> </p>
  • <hn> </hn> con n entre 1 y 6)
  • <header> </header>

La lista completa con sus usos se puede encotrar aquí. https://developer.mozilla.org/es/docs/Web/HTML/Block-level_elements

¿Qué son los elementos de línea?

Son aquellos elementos que solo hacen uso del espacio que están ocupando realmente. Si se trata de un renglón con algún texto, por ejemplo, este elemento no va a autocompletar lo que está a su derecha ni un espacio arriba o abajo, va a ocupar solo lo que le corresponde.

Etiquetas que se comportan como elemento de línea

  • <em> </em>
  • <span> </span>
  • <strong> </strong>

La lista completa con sus usos se puede encontrar aquí. https://developer.mozilla.org/es/docs/Web/HTML/Inline_elements
